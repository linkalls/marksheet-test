# Marksheet Expo App 📱

マークシート試験の作成、配布、採点を可能にする強力なAI駆動モバイルアプリケーションです。React Native、Expo、およびOpenAIで構築されています。

## ✨ 機能

- **🤖 AI試験作成**: OpenAI Visionを使用して、PDFや画像から試験設定を即座に生成します。
  - 自動採点設定のための「問題」と「解答」の同時アップロードをサポート。
  - 複雑な問題形式（例：「AからD」、「1〜5」）に対応。
- **📄 PDFエクスポート**: 印刷可能なマークシートPDFをアプリから直接生成します。
- **📷 AIビジョン採点**: 記入済みの解答用紙をカメラでスキャンまたはインポートして、即座に採点します。
- **⚙️ カスタマイズ可能な設定**: OpenAI APIキー、モデル（例：`gpt-5-mini-2025-08-07`）、デバッグモードを安全に管理します。
- **💾 ローカル保存**: 試験の下書きと採点履歴を自動保存します。

## 🚀 クイックスタート

1. **依存関係のインストール**:

   ```bash
   bun install
   ```

2. **設定**:
   - このアプリはビジョンタスクにOpenAIを使用します。
   - APIキーはアプリ内の**設定**タブで設定するか、環境変数 `.env` ファイルを使用できます（`docs/setup.md`を参照）。

3. **アプリの実行**:
   ```bash
   bun start
   ```
