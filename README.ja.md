# Marksheet Expo App 📱

マークシート試験の作成、配布、採点を可能にする強力なAI駆動モバイルアプリケーションです。React Native、Expo、およびOpenAIで構築されています。

## ✨ 機能

### コア機能
- **🤖 AI試験作成**: OpenAI Visionを使用して、PDFや画像から試験設定を即座に生成します。
  - 自動採点設定のための「問題」と「解答」の同時アップロードをサポート。
  - 複雑な問題形式（例：「AからZ」、「1〜26」）に対応。
  - 複数の解答形式をサポート：アルファベット（A-Z）、数字（1-26）、かな、いろは。
  - 設定可能な選択肢数：問題ごとに2〜26個の選択肢。
- **📄 PDFエクスポート**: 印刷可能なマークシートPDFをアプリから直接生成します。
- **📷 AIビジョン採点**: 記入済みの解答用紙をカメラでスキャンまたはインポートして、即座に採点します。
  - AIビジョン技術による自動採点。
  - エッジケースに対する手動修正サポート。
  - すべて選択問題の複数回答サポート。
- **📊 試験分析**: スコア分布と問題難易度分析を含む詳細な統計を表示します。
- **📦 バッチ操作**: 試験の複製、結果のCSVエクスポート、複数の試験の効率的な管理。
- **⚙️ カスタマイズ可能な設定**: OpenAI APIキー、モデル（例：`gpt-5-mini-2025-08-07`）、デバッグモードを安全に管理します。
- **💾 ローカル保存**: 試験の下書きと採点履歴を自動保存します。
- **🔍 検索とフィルター**: 組み込みの検索機能で保存された試験をすばやく見つけます。

## 🚀 クイックスタート

1. **依存関係のインストール**:

   ```bash
   bun install
   ```

2. **設定**:
   - このアプリはビジョンタスクにOpenAIを使用します。
   - APIキーはアプリ内の**設定**タブで設定するか、環境変数 `.env` ファイルを使用できます（`docs/setup.md`を参照）。

3. **アプリの実行**:
   ```bash
   bun start
   ```

## 💰 コスト見積もり

このアプリはAI機能にOpenAIのAPIを使用します。典型的なコストの見積もりは次のとおりです：

- **GPT-4o Vision**（推奨）：試験生成または採点操作あたり約$0.01-0.05
- **GPT-4o-mini**：操作あたり約$0.001-0.005（より経済的、わずかに精度が低い）

30人のクラスで週1回試験の場合：
- 試験生成：週約$0.05
- 30枚の採点：週約$0.30-1.50
- **合計**：使用量とモデルの選択により年間約$15-70

現在の料金については、[OpenAI Pricing](https://openai.com/pricing)をご覧ください。

## 🎯 使用例

- **教師**: 選択式および短答式試験を効率的に作成および採点。
- **家庭教師**: 迅速な評価作成と学生への即座のフィードバック。
- **教育機関**: 自動採点による標準化されたテスト。
- **オンラインコース**: AI搭載評価によるリモート試験管理。

## ⚠️ 制限事項

- AI機能にはアクティブなインターネット接続が必要です。
- OpenAI APIキーが必要です（含まれていません）。
- 明確で高コントラストのスキャンされた解答用紙で最良の結果が得られます。
- 最適なパフォーマンスのため、試験あたり最大100問を推奨します。
- 現在、単一ページの試験のみをサポートしています。
- 選択式問題あたり最大26個の選択肢をサポートしています。
